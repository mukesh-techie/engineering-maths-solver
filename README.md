# Engineering Maths Solver 🎯

A simple tool to help engineering students 
solve maths problems easily!

## Features
- Quadratic equations solver
- Matrix operations
- Derivative calculator

## How to Use

1. Clone this repository
git clone https://github.com/yourusername/engineering-maths-solver

2. Run the program
python solver.py

3. Enter your values and get answers!

## Who is this for?
- Engineering students
- VIT, Sathyabama, CIT students
- Anyone learning engineering maths
